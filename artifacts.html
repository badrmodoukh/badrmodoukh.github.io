Dynamic Item Templates

One of the major projects I worked on while working at eBay Canada, was creating a dynamic template that pulls items listed on the eBay Canada website dynamically. The item information is pulled in a JSON format and creates an image using HTML and CSS to display the image information. The image includes the item title, item price, whether the item has Free Shipping, and the discount amount on the item. This dynamic item template was used in all major email marketing communications and was sent to over a million users. This was a major accomplishment for me as I worked on the back-end, coding the template, and also the front-end designing how the generated image will look.

In order to get the template to work and display the imported JSON information I had to learn a new template language called FTL (FreeMaker Template Language). FTL is a template engine that is a Java Library and is used to generate text output based on templates and changing data.

Technologies Used: HTML, CSS, JSON, FTL

Christmas Gift Finder Tool

For the Christmas holiday I created a tool that helps buyers shopping on the eBay website find the gifts they want to purchase faster. The tool asks the user to select the type of person (ex. Boy, Girl, Teen, Kids, Pets, etc.), their interest, and the price range they want to spend. Once the user selects their option a unique URL gets generated and displays different items on the eBay website that meets their selected criteria. This was an accomplishing project to work on because I coding everything from the front-end to the back-end of this tool. Once the tool was completed it was used on the eBay Canada website for users to use in the holiday christmas season.

Technologies Used: HTML, CSS, JavaScript, jQuery
